## Создание коммита
Для создания коммита используется комманда *git commit*. Для этого в терминале с папкой репозитория необходимо написать *git commit -m <сообщение к коммиту>*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО!!!***.
